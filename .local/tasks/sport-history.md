# История тренировок и статистика

## What & Why
Создать подраздел «Занятия» (история тренировок) с детальной статистикой, возможностью редактирования прошлых тренировок, и ИИ-отчётами по каждой сессии. Это позволяет пользователю отслеживать прогресс и получать персонализированные рекомендации.

## Done looks like
- Большая кнопка «Заниматься» вверху страницы (ведёт на вкладку активной тренировки)
- Блок статистики: сколько тренировок всего, в какие дни недели чаще всего, в какое время суток, средний % выполнения программы, общая средняя оценка тренировок
- Графики: частота тренировок по неделям, распределение по дням, тепловая карта активности
- Ежедневный ИИ-совет на основе истории тренировок (генерируется при заходе на страницу, кешируется на сутки)
- Список прошлых тренировок (сессий) с датой, программой, процентом выполнения, оценкой
- Открытие прошлой тренировки: детальный просмотр всех упражнений, подходов, результатов
- Редактирование результатов прошлой тренировки (вес, повторения, комментарий) — чтобы исправить ошибки ввода
- ИИ-отчёт по каждой тренировке: анализ результатов в контексте общей истории, рекомендации, прогресс
- Удаление старой тренировки из workouts (миграция на новую систему session-based)

## Out of scope
- Аналитика как отдельный подраздел (будущая работа)
- База упражнений
- Персональный ИИ-тренер (отдельный чат)
- Экспорт данных

## Tasks
1. **API для истории** — GET /api/workout-sessions (с пагинацией и фильтрами), GET /api/workout-sessions/:id (детали с сетами), PATCH /api/workout-sessions/:id (редактирование результатов), DELETE /api/workout-sessions/:id.

2. **API для статистики** — GET /api/workout-sessions/stats: общее количество, частота по дням недели, частота по часам, средний процент выполнения, средняя оценка. Расчёт на стороне сервера.

3. **API для ИИ-совета и отчётов** — GET /api/workout-sessions/daily-advice: ИИ генерирует совет дня на основе последних тренировок (кешируется). POST /api/workout-sessions/:id/ai-report: ИИ анализирует конкретную тренировку и историю, составляет отчёт.

4. **Фронтенд — страница «Занятия»** — Верхний блок с кнопкой «Заниматься» и статистическими карточками. Графики активности (Recharts). Блок ИИ-совета дня. Список прошлых тренировок с возможностью раскрытия деталей.

5. **Фронтенд — просмотр/редактирование сессии** — Модальное окно или отдельная страница: все упражнения и подходы, возможность редактирования каждого результата, кнопка «Получить ИИ-отчёт», отображение отчёта.
