# Активная тренировка — пошаговое выполнение

## What & Why
Реализовать ядро тренировочного процесса — экран активной тренировки, где пользователь пошагово выполняет упражнения из выбранной программы. Разные типы тренировок (зал, бег, плавание) имеют разные интерфейсы и логику. После завершения ИИ анализирует результаты и предлагает адаптацию нагрузки.

## Done looks like
- Кнопка «Заниматься» из любого места открывает экран выбора программы (по умолчанию — основная), с галочкой «сделать основной»
- Для тренировок в зале (сетами): показывается одно упражнение, техника выполнения, советы по текущему подходу (сгенерированы при создании программы), советы по весам и повторениям (на основе истории, если есть), показ следующего упражнения в мини-окне
- Если это первая тренировка по программе — сообщение что совет по весам будет после первого занятия
- Кнопки: «Выполнено» (→ ввод веса, повторений, смайлик оценки), «Пропустить подход» (→ следующий подход/упражнение), «Пропустить упражнение» (→ следующее упражнение)
- Для плавания: показ всей программы сразу (без телефона в бассейне), после тренировки — ввод всех результатов разом
- Для бега: трекинг по времени/дистанции, контрольные точки (интервалы)
- Для йоги/растяжки: таймер по каждой позе, автопереход
- После завершения всей программы: процент выполнения, возможность оставить комментарий, ИИ анализирует результаты тренировки в контексте истории и может предложить увеличить/уменьшить нагрузку — пользователь может согласиться или оставить как есть
- Сохранение сессии тренировки со всеми результатами в БД

## Out of scope
- История тренировок и отчёты (следующая задача)
- Таймер с GPS-трекингом бега (ограничение веб-приложения)
- Аналитика, база упражнений, персональный ИИ-тренер

## Tasks
1. **Экран выбора программы** — Список программ пользователя, основная выделена и выбрана по умолчанию, галочка «сделать основной» при выборе другой. Кнопка «Начать тренировку» → создание сессии в БД.

2. **Режим тренировки «Сеты» (зал)** — Полноэкранный режим тренировки: текущее упражнение крупно (название, техника, советы), номер подхода, превью следующего упражнения. Кнопки: «Выполнено» (модальное окно — вес, повторения, смайлик), «Пропустить подход», «Пропустить упражнение». Прогресс-бар общий. Советы по весам на основе предыдущих тренировок (если есть).

3. **Режим тренировки «Обзор» (плавание)** — Вся программа показывается списком перед тренировкой. После нажатия «Завершить тренировку» — форма для занесения всех результатов разом (дистанция, время, примечания по каждому элементу).

4. **Режим тренировки «Время/Дистанция» (бег, велосипед)** — Интерфейс с таймером, отметки по интервалам/контрольным точкам. Запись времени и дистанции каждого сегмента.

5. **Режим тренировки «Таймер» (йога, растяжка)** — Таймер для каждой позы/упражнения, автоматический переход к следующему, звуковой сигнал.

6. **Экран завершения тренировки** — Сводка результатов: выполнено/пропущено, общий процент, время. Поле для комментария. ИИ-анализ с предложением адаптации нагрузки (увеличить/уменьшить). Кнопки «Принять изменения» / «Оставить как есть». Сохранение финальных данных.

7. **API для активной тренировки** — POST /api/workout-sessions/start, PATCH /api/workout-sessions/:id/log-set, PATCH /api/workout-sessions/:id/complete, POST /api/workout-sessions/:id/ai-analysis. Получение предыдущих результатов для советов по весам.
